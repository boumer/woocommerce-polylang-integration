Polylang and Woocommerce Integration

== Description ==
Simple plugin that integrates the popular ecommerce plugin for Wordpress called Woocommerce with Polylang plugin for Wordpress

== Installation ==
It\'s fairly simple, please follow the steps

    1. Make sure you have installed Polylang and Woocommerce
    2. Install Woocommerce-Polylang-Integration from this repo
    3. Go to Dashboard > Settings > Languages > Settings and mark the checkbox for Custom post types => Products
    4 Add your products in your languages using the \"plus\" icon in Dashboard > Products
    and done!

If you have any question feel free to get in touch

== Frequently Asked Questions ==
Why was this plugin created?
I got a lot multilingual Projects that involves Woocommerce and my best open source alternative is using Polylang. So i did this plugin to make Woocommerce work in combination with Polylang
